![task-manager](https://github.com/user-attachments/assets/068ca35d-852f-48a5-8139-2d26cef1a278)


## 🚀 Tech Stack

- **Runtime**: Node.js v20+
- **Framework**: Express.js
- **Language**: TypeScript
- **Database**: PostgreSQL
- **ORM**: Prisma
- **Auth**: JWT
- **API Docs**: Swagger UI

## 📦 Prerequisites

- Node.js v20+
- PostgreSQL 14+
- npm/yarn/pnpm

## 🛠️ Installation

```bash
git clone https://github.com/your-repo/task-manager-backend.git
cd task-manager-backend
npm install
```
